: Введение в Tact и основы TON

Что такое Tact и TON

Tact - это высокоуровневый язык программирования для разработки смарт-контрактов на блокчейне TON (The Open Network).
TON - это многоуровневый блокчейн с уникальной архитектурой:

Мастерчейн: основная координирующая цепь
Воркчейны: отдельные блокчейны с собственными правилами
Шардчейны: подразделения воркчейнов для масштабируемости

Базовые типы данных в Tact

Int: целые числа (257-бит)
Bool: логические значения (true/false)
Address: адреса контрактов и кошельков
Cell: базовая структура данных для хранения информации
String: текстовые строки

Структура смарт-контракта

tactCopycontract MyContract {
    // Переменные состояния
    owner: Address;
    counter: Int as uint32;

    // Конструктор
    init() {
        self.owner = sender();
        self.counter = 0;
    }

    // Получатель (обработчик входящих сообщений)
    receive("increment") {
        self.counter = self.counter + 1;
    }

    // Геттер (метод для чтения данных)
    get fun value(): Int {
        return self.counter;
    }
}

Особенности TON и Tact

Асинхронная модель: контракты взаимодействуют через сообщения
Газ: оплата вычислительных ресурсов
Шардинг: высокая масштабируемость
Персистентное хранение: оплата за хранение данных

Базовые операции

Отправка сообщений: send()
Проверка условий: require()
Логирование: dump()
