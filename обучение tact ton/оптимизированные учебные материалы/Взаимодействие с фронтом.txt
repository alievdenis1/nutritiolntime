TON Connect 2.0

Стандартный протокол для подключения кошельков к dApps
Позволяет пользователям безопасно подписывать транзакции через поддерживаемые кошельки


TonWeb.js

JavaScript библиотека для взаимодействия с блокчейном TON
Поддерживает создание транзакций, работу с адресами и взаимодействие со смарт-контрактами


@ton-community/ton-api

TypeScript SDK для работы с TON API
Предоставляет удобные методы для получения данных из блокчейна


ton-x

SDK для разработки TON dApps
Включает инструменты для работы с контрактами и кошельками


ton-core

Низкоуровневая библиотека для работы с TON
Предоставляет базовые примитивы и функции для взаимодействия с блокчейном


@twa-dev/sdk

SDK для разработки Telegram Web Apps (TWA) на TON
Позволяет создавать приложения, интегрированные с Telegram


TON Access

Сервис для доступа к TON API через HTTP
Упрощает интеграцию TON в веб-приложения


@orbs-network/ton-access

JavaScript SDK для работы с TON Access
Предоставляет простой интерфейс для взаимодействия с TON из браузера


tonweb-mnemonic

Библиотека для работы с мнемоническими фразами в TON
Полезна при реализации функций восстановления кошелька
